## 如何部署到GitHub

### 方法1：手动上传
1. 访问 https://github.com/SunsetzF2023/Wcardpve
2. 点击 "Add file" → "Upload files"
3. 将所有文件拖拽上传
4. 提交更改

### 方法2：使用Git命令行
```bash
# 在项目目录中执行
git init
git add .
git commit -m "Initial commit: 文字对战扑卡牌游戏"
git remote add origin https://github.com/SunsetzF2023/Wcardpve.git
git push -u origin main
```

## 启用GitHub Pages

1. 进入GitHub仓库
2. 点击 "Settings" 选项卡
3. 找到 "Pages" 部分
4. 在 "Source" 中选择 "Deploy from a branch"
5. 选择 "main" 分支和 "/ (root)" 文件夹
6. 点击 "Save"

几分钟后，您的游戏就可以通过以下URL访问：
`https://sunsetzf2023.github.io/Wcardpve/`
